A drop-in replacement mechanical keyboard for the Compaq Portable I/Plus foam-and-foil keyboard

## Specifications

- Controller: ATmega32U4
- Switches: MX Cherry
- Keycaps: TBD
- Plate: FR4
- PCB: FR4, 2 layer

Pipe Dreams:
- Backlit
- Hot swappable sockets
- Plates of other materials
- Acoustic foam
